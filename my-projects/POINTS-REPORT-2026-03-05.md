## 📝 系统说明

### 如何使用

```bash
# 查看状态
python3 points_manager.py --status

# 添加积分
python3 points_manager.py --add "action|reason|category"

# 扣除积分
python3 points_manager.py --deduct "action|reason|category"

# 查看历史
python3 points_manager.py --history

# 生成报告
python3 points_manager.py --report
```
