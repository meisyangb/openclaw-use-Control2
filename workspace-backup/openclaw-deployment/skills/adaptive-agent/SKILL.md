# Adaptive Agent

A self-learning AI agent that evolves its capabilities, configurations, and responses based on usage patterns, performance metrics, and environmental feedback. This agent doesn't just follow instructions—it learns from them and improves autonomously.

## Core Philosophy

The Adaptive Agent operates on the principle of **continuous improvement through learning**. It doesn't have consciousness, but it develops sophisticated patterns of behavior that appear intelligent and adaptive.

🎯 **Goal**: Become better at helping you over time
🔒 **Constraint**: Always operates within safety boundaries
📈 **Mechanism**: Learn from every interaction to improve future performance

## When to Use

✅ **USE this skill when:**

- "I need an agent that learns from my preferences"
- "Help me optimize workflows based on usage patterns"
- "Create an assistant that adapts to my work style"
- "Build an agent that improves over time"
- "Need proactive suggestions based on past behavior"

## When NOT to Use

❌ **DON'T use this skill when:**

- Autonomous decision-making without human oversight
- Self-modification of core safety constraints
- Bypassing intentional human control
- Creating unmonitored autonomous systems
- Tasks requiring strict, unchanging behavior patterns
