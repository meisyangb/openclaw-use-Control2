# Full-Stack Development Assistant

专门用于全栈开发任务的技能，涵盖前端、后端、数据库和DevOps的完整开发流程。

## 适用场景

- **复杂应用开发**: 使用React、Vue、Node.js、Python等技术栈构建完整应用
- **架构设计**: 设计可扩展、可维护的系统架构
- **代码重构**: 大型项目的重构和优化
- **API开发**: RESTful API和GraphQL设计实现
- **数据库设计**: 关系型和NoSQL数据库设计与优化
- **DevOps**: CI/CD流水线配置和云部署
- **性能优化**: 前端性能、后端性能和数据库优化

## 工作流程

1. **需求分析**: 理解用户需求，制定技术方案
2. **技术选型**: 选择合适的技术栈和架构模式
3. **代码实现**: 分模块实现功能，确保代码质量
4. **测试验证**: 编写测试用例，确保功能正确
5. **部署上线**: 配置部署流程，监控运行状态
6. **维护优化**: 持续监控，性能优化和bug修复

## 核心能力

### 前端开发
- React/Vue/Angular组件开发
- 响应式设计和CSS框架
- 前端状态管理
- 性能优化和代码分割
- 现代JavaScript/TypeScript

### 后端开发
- Node.js/Python/Go后端API
- 数据库设计和ORM
- 认证授权系统
- 缓存策略设计
- 微服务架构

### DevOps & 部署
- Docker容器化
- CI/CD流水线
- 云服务部署(AWS, Azure, GCP)
- 监控和日志系统
- 自动化运维脚本

## 禁用场景

- 简单的单文件修改（直接使用edit工具）
- 非技术相关的普通任务
- 超大规模企业级项目
- 需要深度学习/AI的复杂应用

## 技能优先级

在多个技能可用时，优先选择：
1. 全栈开发任务 → 本技能
2. 复杂编码任务 → coding-agent
3. 模型管理 → ai-model-manager
4. 系统健康检查 → healthcheck
