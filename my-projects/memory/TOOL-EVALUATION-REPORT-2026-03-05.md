## 📋 评估标准

### 评分计算

```
综合评分 = ROI×0.4 + 效率×0.3 + 满意度×0.3

其中:
- ROI = 时间节省 / 时间投入
- 效率 = 平均每次使用节省的时间 / 平均每次使用时间
- 满意度 = 主观评分 0-1
```

### 推荐操作

| 评分范围 | 推荐 | 操作 |
|---------|------|------|
| ≥ 0.8 | ✅ KEEP | 保留并继续使用 |
| 0.5-0.8 | 🔧 IMPROVE | 改进后使用 |
| 0.3-0.5 | ⚠️ REVIEW | 审查决定 |
| < 0.3 | ❌ DEPRECATE | 废弃 |
